# Contribuir a la SPA de Catálogo de Videojuegos

¡Gracias por tu interés en contribuir a este proyecto! Damos la bienvenida a contribuciones de todos.

## Cómo Contribuir

1.  **Hacer un Fork del repositorio**
    Haz clic en el botón "Fork" en la parte superior derecha de esta página para crear tu propia copia del repositorio.

2.  **Clonar tu fork**

    ```bash
    git clone https://github.com/TU_USUARIO/SPA.git
    cd SPA
    ```

3.  **Crear una rama (branch)**
    Crea una nueva rama para tu funcionalidad o corrección de errores.

    ```bash
    git checkout -b feature/funcionalidad-increible
    ```

4.  **Hacer tus cambios**
    Implementa tu funcionalidad o corrige el error. Asegúrate de que tu código siga el estilo existente (Vanilla JS, componentes funcionales).

5.  **Hacer commit de tus cambios**
    Escribe mensajes de commit claros y concisos.

    ```bash
    git commit -m "Agregar alguna funcionalidad increíble"
    ```

6.  **Hacer Push a tu fork**

    ```bash
    git push origin feature/funcionalidad-increible
    ```

7.  **Abrir un Pull Request**
    Ve al repositorio original y abre un Pull Request desde tu rama. Describe tus cambios y por qué son necesarios.

## Estilo de Código

- Usa sintaxis **ES6+**.
- Mantén los componentes pequeños y enfocados.
- Usa `async/await` para operaciones asíncronas.
- Formatea tu código de manera consistente (ej. usando Prettier).

## Reportar Problemas

Si encuentras un error o tienes una solicitud de funcionalidad, por favor abre un "issue" en el repositorio. Proporciona tanto detalle como sea posible, incluyendo pasos para reproducir el problema.
